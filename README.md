# TechLearn
TechLearn Hub — A full-stack educational web app built with Python &amp; Django.


📚 Learn 10 programming languages (Python, Django, Java, Spring Boot, HTML, CSS, JavaScript, React, Node.js, SQL) through curated tutorials and test your knowledge with interactive quizzes.

✨ Features:
- 🔐 Role-based authentication (Normal User & Admin)
- 🧠 50+ MCQ quizzes with real-time green/red feedback
- 🤖 Career-based learning roadmap recommendations
- 📊 Admin panel with user management & quiz statistics
- 🎨 Dark glassmorphism UI — no CSS framework used

🛠️ Tech Stack: Python · Django 5.0 · SQLite · HTML5 · CSS3 · Vanilla JavaScript · Django ORM
